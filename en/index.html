Trust

Mandatory parts of the approved work and review flow.

Professional duties and privacy

Website data is covered by the privacy notice. The professional privacy and duty assumptions live in the GitHub reference for each approved work and review flow.

Human in the loop is mandatory

Every productive approved work and review flow includes human review, approval and ownership. Automation prepares; it does not replace notarial judgment.

Documentation is mandatory

Every test or productive run needs flow version, decision points, approvals and justified deviations. Without that evidence, the flow is incomplete.

Organizational separation

The application is designed for organizational separation: work areas, responsibilities, data, approvals, and working states are kept separate by organization.

Software components remain attributable

When a Software Bill of Materials (SBOM) is maintained and reviewed for an approved version, it assigns components, versions, dependencies, security notices and license questions to that version.
