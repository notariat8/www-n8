Positioning

More digital support without losing control.

Many notarial offices want digital support, but not a black box. Notariat8 combines modern assistance with clear professional control: standards stay visible, responsibilities remain understandable and sensitive information does not belong on a public website.

Where Notariat8 helps.

Use cases

Use cases as final workflows.

Self-contained use cases

Each use case stands on its own. Notariat8 provides scope, roles, BPMN modeling, review points and mandatory documentation as a final workflow.

Parallel operation is possible

A use case can run alongside the existing way of working. Value appears fastest when the 10 most important use cases are implemented digital first.

GitHub as the reference

The GitHub reference describes the current final workflow. Git audit and PRs show how it improves over time.

Justify deviations

Local adaptations are possible. The question is: what is so specific that everyone would not benefit from a standard that becomes more standardized and better over time?

Trust

Mandatory parts of the final workflow.

Professional duties and privacy

Website data is covered by the privacy notice. The professional privacy and duty assumptions live in the GitHub reference for each final workflow.

Human in the loop is mandatory

Every productive final workflow includes human review, approval and ownership. Automation prepares; it does not replace notarial judgment.

Documentation is mandatory

Every test or productive run needs workflow version, decision points, approvals and justified deviations. Without that evidence, the flow is incomplete.

n8, GitHub and app

n8 is the website with the public process view. GitHub is the reference. app.notariat8.de is the test and execution surface; validation runs there, not on this website.
